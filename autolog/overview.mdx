## What is Auto Log?

`autoLog()` is the core function of `@vita-tak/ai-logger`. It wraps an AI client and automatically logs every inference call — without requiring any changes to your existing code logic.

Every logged call is cryptographically stamped, secured and verified on the Minima blockchain, giving you a tamper-proof audit trail, tailored for EU AI Act compliance.

Your code receives the response as normal. The logging happens asynchronously in the background and never blocks your application.
