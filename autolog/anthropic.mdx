## Overview

Wrapping an Anthropic client works identically to OpenAI. The provider is auto-detected from the client instance — no manual configuration needed.

## Usage
```typescript
import { AILogger } from '@vita-tak/ai-logger'
import Anthropic from '@anthropic-ai/sdk'

const aiLogger = new AILogger({
  apiKey: process.env.INTEGRITAS_API_KEY,
  systemId: 'your-system-name',
})

const client = aiLogger.autoLog(new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY }))

const response = await client.messages.create({
  model: 'claude-3-5-sonnet-20241022',
  max_tokens: 100,
  messages: [{ role: 'user', content: 'Hello!' }]
})

// ✓ Logged & stamped on blockchain automatically
```

<Warning>
  Always use the wrapped `client` variable for your calls — not the original `anthropic` instance. Calls made through the unwrapped instance will not be logged.
</Warning>

## Works with any setup

Just like with OpenAI, it does not matter how the surrounding code is structured. As long as every AI call passes through the wrapped client, it will be captured and logged.
```typescript
// Next.js API route — works fine
export async function POST(req: Request) {
  const { message } = await req.json()

  const response = await client.messages.create({
    model: 'claude-3-5-sonnet-20241022',
    max_tokens: 100,
    messages: [{ role: 'user', content: message }]
  })

  return Response.json({ reply: response.content[0].text })
}
```

## Works with AI agents too

`autoLog()` captures every call the agent makes — regardless of which tool or step triggers it.
```typescript
const response = await client.messages.create({
  model: 'claude-3-5-sonnet-20241022',
  tools: [...],
  messages: [...]
})

// ✓ All agent calls logged automatically
```

<Info>
  This applies to custom-built agents and simple orchestration logic. For LangChain, see the dedicated guide.
</Info>
