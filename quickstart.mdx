title: "Quick Start"
description: "Integrate @vita-tak/ai-logger in three steps."
---

## Step 1 — Install
```bash
npm install @vita-tak/ai-logger
```

## Step 2 — Initialize the logger
```typescript
import { AILogger } from '@vita-tak/ai-logger'
import OpenAI from 'openai'

const aiLogger = new AILogger({
  apiKey: process.env.INTEGRITAS_API_KEY,
  systemId: 'your-system-name',
})
```

## Step 3 — Wrap your AI client with autoLog()
```typescript
const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY })

// Wrap the client — this is the only change needed
const client = aiLogger.autoLog(openai)

// Use 'client' exactly as you would use 'openai'
const response = await client.chat.completions.create({
  model: 'gpt-4o-mini',
  messages: [{ role: 'user', content: 'Hello!' }]
})

// ✓ Response logged & stamped on blockchain automatically
```

<Warning>
  Always use the wrapped `client` variable for your calls — not the original `openai` instance. Calls made through the unwrapped instance will not be logged.
</Warning>

## Full example
```typescript
import { AILogger } from '@vita-tak/ai-logger'
import OpenAI from 'openai'
import 'dotenv/config'

const aiLogger = new AILogger({
  apiKey: process.env.INTEGRITAS_API_KEY,
  systemId: 'my-ai-system',
})

const client = aiLogger.autoLog(new OpenAI({ apiKey: process.env.OPENAI_API_KEY }))

const response = await client.chat.completions.create({
  model: 'gpt-4o-mini',
  max_tokens: 100,
  messages: [{ role: 'user', content: 'Hello!' }]
})

console.log(response.choices[0]?.message.content)
// ✓ Logged & stamped automatically
```
